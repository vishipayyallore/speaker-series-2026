This repository is the working space for planning, drafting, and organizing Speaker Series 2026 materials.

---

## Purpose

The project is intended to centralize assets and working files for the 2026 speaker program, including:

- session planning and outlines
- speaker and topic documentation
- supporting visuals and media
- implementation or automation files as needed

---

## Planned Content Areas

As work progresses, this repository can be expanded with sections such as:

- `src/sessions/` for session-specific plans
- `src/speakers/` for speaker profiles and notes
- `src/wbs-scenarios/` for work-breakdown or scenario planning
- `docs/` for operational and process documentation

The exact structure will be finalized as the Speaker Series plan is approved.

---

## Working Guidelines

- Keep documentation concise and versioned in this repository.
- Place reusable assets under `docs/images/`.
- Keep source artifacts and automation scripts under `src/`.
- Prefer clear file naming for fast discovery during event execution.
