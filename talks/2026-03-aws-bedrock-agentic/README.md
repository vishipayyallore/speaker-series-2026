# AWS Bedrock + Agentic Coding

**Event:** Dot Net Learners House  
**Date:** 2026-03  
**Duration:** 60 minutes  
**Status:** planned

---

## Abstract

Hands-on introduction to Amazon Bedrock for generative AI workloads, combined with agentic coding patterns: tool use, routing prompts, and a reproducible local demo attendees can run from this repository.

---

## Audience

Developers familiar with Python or any modern language who want a practical Bedrock and agents walkthrough.

---

## Prerequisites

- AWS account with Bedrock model access enabled
- Python 3.11+ and Docker (optional, for local stack)
- Copy `.env.example` from repo root to `.env` and configure credentials
