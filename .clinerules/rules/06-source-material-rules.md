# External curriculum rules

## Python hands-on content

All Python lab code and session docs live in:

**[python-fundamentals-in-practice](https://github.com/vishipayyallore/python-fundamentals-in-practice)**

This repo stores only:

- talk index stub (`talks/NN-l1-sN-short-name/README.md`)
- links to `docs/sessions/L1/S5.md`, `S6.md`, etc.
- clone/run commands for attendees

Do **not** copy session markdown or `src/L1/` code here.

## Adding a delivered Python session

1. Create `talks/NN-l1-sN-short-name/README.md` (next sequence number in delivery order)
2. Link session doc and practice folder on GitHub
3. Update root README completed table
4. Mark status `delivered` in talk README

## In-repo demos

Bedrock, Cline, and similar sessions keep full materials under their talk folder.
